# Tool Categories and Likely Locations

## Electronics Category
**Tools**: desklamp, remotecontrol, cellphone, laptop, tablet, television
**Likely Locations**:
- Primary: sidetable, desk, shelf
- Secondary: drawer, cabinet
- Tertiary: sofa, armchair, bed

**Search Priority**: 
1. Check all sidetables in the room
2. Check desks and shelves
3. Check drawers near seating areas
4. Check other surfaces

## Kitchen Tools Category
**Tools**: knife, spoon, fork, plate, bowl, cup, mug, pan, pot, spatula
**Likely Locations**:
- Primary: countertop, cabinet
- Secondary: fridge, microwave, drawer
- Tertiary: diningtable, shelf

**Search Priority**:
1. Check kitchen countertops
2. Check cabinets (especially lower ones)
3. Check drawers labeled for utensils
4. Check fridge for food-related items

## Cleaning Tools Category
**Tools**: sponge, cloth, towel, broom, mop, spraybottle, soap
**Likely Locations**:
- Primary: shelf, cabinet
- Secondary: countertop, drawer
- Tertiary: garbagecan, sink

**Search Priority**:
1. Check cleaning supply shelves
2. Check under-sink cabinets
3. Check utility drawers
4. Check near sinks and garbage cans

## Reading/Writing Category
**Tools**: book, newspaper, magazine, pen, pencil, paper
**Likely Locations**:
- Primary: sidetable, desk, shelf
- Secondary: drawer, cabinet
- Tertiary: sofa, armchair

**Search Priority**:
1. Check desks and sidetables
2. Check bookshelves
3. Check drawers in study areas
4. Check seating areas

## General Search Heuristics
1. **Context Matters**: Tools are usually stored near where they're used
2. **Size Considerations**: Small tools on surfaces, larger tools in storage
3. **Frequency of Use**: Frequently used tools are more accessible
4. **Room Type**: Different rooms have different tool distributions

## Cross-Category Tools
Some tools may belong to multiple categories:
- **Scissors**: Could be in kitchen (utensil drawer) or office (desk drawer)
- **Flashlight**: Could be electronics (sidetable) or utility (drawer)
- **Keys**: Could be on sidetable, in drawer, or on hook

## Search Fallback Strategy
When uncertain about tool category:
1. Check all sidetables and surfaces
2. Check all drawers
3. Check all shelves
4. Check all cabinets
5. Check furniture surfaces (sofa, armchair, bed)
