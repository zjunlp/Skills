# Action Primer for ScienceWorld Environment

## Relevant Actions for This Skill
- `teleport to <LOC>`: Move instantly to a named room.
- `look around`: Observe the current room and its contents.
- `pick up <OBJ>`: Take an object into your inventory.
- `use <OBJ> [on <OBJ>]`: Employ a tool (e.g., thermometer) on a target.
- `move <OBJ> to <OBJ>`: Transfer an object (from inventory) to a container.
- `focus on <OBJ>`: Signal intent on a task-relevant object.

## Key Object Types
- **Measurement Tools:** Objects like `thermometer` that yield a numeric reading.
- **Target Substances:** Objects like `unknown substance B` that require measurement.
- **Destination Containers:** `green box` and `blue box` for conditional placement.

## Common Room Names
- kitchen, foundry, workshop, bathroom, outside, living room, bedroom, greenhouse, art studio, hallway
