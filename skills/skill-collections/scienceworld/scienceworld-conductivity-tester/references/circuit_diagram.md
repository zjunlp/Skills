# Test Circuit Schematic

## Component List (Found on Workshop Table)
1.  **Battery:** Power source. Has an *anode* (+) and *cathode* (-).
2.  **Wires:**
    - **Orange Wire:** Connects battery anode to light bulb cathode.
    - **Yellow Wire:** Connects battery cathode to the target object.
    - **Green Wire:** Connects the target object to the light bulb anode.
3.  **Light Bulbs:** Use the **blue light bulb** as the indicator.
4.  **Target Object:** Must have two connection points (terminal 1 & terminal 2).
