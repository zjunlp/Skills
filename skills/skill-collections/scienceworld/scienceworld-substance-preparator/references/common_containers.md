# Common Container Reference

## For Heating/Mixing (Primary)
*   `metal pot`
*   `beaker` (glass)
*   `bowl`
*   `cauldron` (foundry)
*   `crucible` (foundry)

## For General Holding (Secondary)
*   `ceramic cup`
*   `glass cup`
*   `tin cup`
*   `wood cup`
*   `glass jar`
*   `flask`

## Typically Not Suitable
*   `drawer` (fixed furniture)
*   `fridge`/`freezer` (appliance, not portable)
*   `sink` (plumbing fixture)
*   `oven` (appliance)

## Verification Commands
*   `look at [CONTAINER]` - Lists contents.
*   `examine [CONTAINER]` - Confirms object type and state.
*   `look around` - Scans room for all available containers.
