# Action Primer for ScienceWorld Environment

## Relevant Actions for Pouring Skill
*   `pour OBJ into OBJ`: The core action. Transfers the contents of the first object (source) into the second object (target).
*   `look around`: Describes the current room and lists all visible objects and containers. Use this to find source and target containers.
*   `examine OBJ`: Provides a detailed description of a specific object, which can help verify its contents or state.
*   `look at OBJ`: Describes a container's contents. Useful for checking what is inside a cupboard or on a table before and after pouring.

## Environment Notes (From Trajectory)
*   All containers are already opened. You can directly interact with items inside them.
*   You can `teleport to LOC` to move between rooms quickly.
*   Common rooms for materials: `art studio` (for paints), `workshop`, `greenhouse`.
*   Common target containers: `jug`, `bowl`, `wood cup`.
