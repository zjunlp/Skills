# Action Primer for ScienceWorld Environment

## Relevant Actions for Substance Fetching
- `teleport to LOC`: Instantly move to a named room (e.g., `kitchen`, `workshop`).
- `look around`: Get a full description of the current room and its contents.
- `examine OBJ`: Get a detailed description of a specific object.
- `look at OBJ`: Describe a container's contents (redundant if `look around` was just used).
- `pick up OBJ`: Move a portable object to your inventory.
- `move OBJ to OBJ`: Move an object to a container or surface.

## Typical Container Names
*   `fridge`
*   `freezer`
*   `cupboard`
*   `counter`
*   `drawer`
*   `shelf`
*   `table`

## Common Substance Locations
*   **Kitchen:** Food items (chocolate, fruit), cooking ingredients (salt, oil).
*   **Workshop:** Chemicals, tools, raw materials.
*   **Greenhouse:** Plants, soil, water.
*   **Art Studio:** Paints, solvents, clay.
