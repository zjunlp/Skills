## Distance Guidelines

### Walking Distance Categories
- **Optimal**: < 500m (5-7 minute walk)
- **Acceptable**: 500m - 1km (7-15 minute walk)
- **Far**: 1km - 2km (15-25 minute walk)
- **Too far**: > 2km (consider alternative transport)

### Radius Search Recommendations
- Urban areas: Start with 500-1000m radius
- Suburban areas: Start with 1000-2000m radius
- Rural areas: Start with 2000-5000m radius

## Output Formatting Standards

### Station Naming
- Use official station names when available
- Include line/network identifiers in parentheses
- Example: "Little India MRT Station (DT12/NE7)"

### Distance Presentation
- Always show both distance and time
- Format: "0.5 km (7 minute walk)"
- Round to nearest 0.1 km for distances, nearest minute for times

### Route Description
- List stations in sequential order
- Indicate transfer points clearly
- Include line changes with direction
- Example: "Transfer to East-West Line heading towards Pasir Ris"
