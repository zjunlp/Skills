- [ ] List workspace directory contents

## Phase 2: Course Search
- [ ] Search for each required course by name
- [ ] Search for course categories (e.g., "思政B", "通识教育核心课程")

## Phase 3: Filtering
- [ ] Apply day of week filter (e.g., Monday only)
- [ ] Apply exam type filter (e.g., open-book only)
- [ ] Check for time conflicts between fixed courses
- [ ] Identify all valid options for variable courses

## Phase 4: Generation

- [ ] Create separate Excel files for each valid scheme
- [ ] Verify file content matches expected format

## Phase 5: Cleanup & Summary
- [ ] Remove temporary/reference files if requested
- [ ] Provide clear summary of created files
- [ ] List fixed courses and variable options
