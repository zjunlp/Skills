# Canvas Integration Workflow

This reference outlines the steps and API calls needed to interact with Canvas LMS for grading.

## Prerequisites
1. **Course ID:** Identify which Canvas course contains the assignment.
2. **Assignment ID:** Identify the specific assignment to grade.
3. **Student Canvas IDs:** Map student identities (from email) to their Canvas user IDs.
