# Common Conference Venue Abbreviations

## AI/ML Conferences
- ICLR: International Conference on Learning Representations
- ICML: International Conference on Machine Learning
- NeurIPS: Conference on Neural Information Processing Systems
- COLM: Conference on Lifelong Learning Agents
- AAAI: Association for the Advancement of Artificial Intelligence
- ACL: Association for Computational Linguistics
- EMNLP: Empirical Methods in Natural Language Processing
- NAACL: North American Chapter of the ACL

## Computer Science
- CVPR: Computer Vision and Pattern Recognition
- ICCV: International Conference on Computer Vision
- ECCV: European Conference on Computer Vision
- SIGGRAPH: Special Interest Group on Computer Graphics
- OSDI: Operating Systems Design and Implementation
- SOSP: Symposium on Operating Systems Principles

## Journal Formats
- Journal of Machine Learning Research (JMLR)
- Transactions on Machine Learning Research (TMLR)
- IEEE Transactions on Pattern Analysis and Machine Intelligence (TPAMI)
- International Journal of Computer Vision (IJCV)

## Formatting Rules
1. Use ALL CAPS for short abbreviations (ICLR, ICML, CVPR)
2. Use Title Case for longer names (Conference on Lifelong Learning Agents)
3. Include year with period: "ICLR 2025."
4. For tracks: "NeurIPS 2023 (Datasets and Benchmarks track)."
5. For arXiv only: "arXiv preprint." or "Preprint."
