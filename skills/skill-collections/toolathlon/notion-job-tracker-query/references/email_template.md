# Job Application Email Template

Use this template when the user requests to send application emails for qualifying job entries.

## Template Variables
- `{Company Name}`: Extracted from the `Company` property.
- `{Position}`: Extracted from the `Position` property.
- `{Your Name}`: The applicant's name (e.g., "Janet Mendoza"). This should be provided by the user or configured.
