# Example Formatting Rules (tips.txt)
This file specifies how to format the data in the Excel output.

1.  **Strictly follow the units in the file.**
    *   Check column headers for units (e.g., "Million Shares", "Billion USD", "%").
    *   Convert raw data to match these units before writing.

2.  **Round all numbers to two decimal places.**
    *   Apply to all numerical values (prices, percentages, calculated figures).

3.  **If the data is unavailable, fill in NaN.**
    *   Use the string "NaN" in the cell if data cannot be retrieved or calculated.
