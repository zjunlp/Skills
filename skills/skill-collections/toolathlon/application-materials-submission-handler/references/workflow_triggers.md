# Trigger Phrases for Application Materials Submission

## Primary Triggers
- "submit application materials"
- "follow email instructions for submission"
- "organize files according to guidelines"
- "send materials to [recipient]"

## Application Type Specific
- "PhD application"
- "job application"
- "grant submission"
- "scholarship application"
- "fellowship application"

## Action-Oriented Requests
- "check my inbox for an email from [sender] about submission"
- "help me submit the relevant materials"
- "package my application files"
- "create the folder structure for my application"
- "merge my award certificates"
- "send my application to [email address]"

## Key Indicators
- References to specific email instructions
- Mentions of folder structure requirements
- Requests involving file organization with naming conventions
- Need to retrieve personal information from memory
- Requirement to send formatted email with attachments
