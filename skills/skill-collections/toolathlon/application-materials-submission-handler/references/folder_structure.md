# Standard Folder Structure for Application Materials

## Root Folder Naming
- Format: `Application_Materials_[YourName]_[YourStudentID]`
- Rules:
  - No spaces in name
  - Use English characters only
  - Replace `[YourName]` with your name without spaces (e.g., "MaryCastillo")
  - Replace `[YourStudentID]` with your actual student ID
